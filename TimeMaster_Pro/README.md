# TimeMaster Pro

Минималистичное приложение: секундомер + таймер.

## Что внутри

- `index.html` — структура приложения
- `style.css` — красивый минималистичный интерфейс
- `script.js` — логика секундомера и таймера

## Как запустить

Открой файл `index.html` в браузере.

Или открой папку в VS Code и запусти через расширение Live Server.
